Week04 Day1 Portfolio / Investment Decisions Continued
This project utilized quantitative and numerical financial data analysis skills. Return indicators were calculated using historical asset prices, key statistical characteristics were analyzed, and investment risk was assessed using standard deviation. Asset relationships were also examined using correlation and covariance measures. The Sharpe ratio was used to evaluate investment performance. This project demonstrates skills in working with financial time series, analyzing risk-return relationships, and presenting results in a structured analytical format.
